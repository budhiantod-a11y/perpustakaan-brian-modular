📌 Cara hosting gratis (GitHub Pages, 5 menit):
1. Buat repo baru di github.com
2. Upload semua file (index.html, style.css, folder js/) ke repo tersebut
3. Buka Settings → Pages, pilih Branch: main, lalu Save
4. Dalam 1-2 menit, site aktif di https://username.github.io/repo-name
